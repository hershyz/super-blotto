# Client Setup

Only a Python 3 install is required — no external packages needed (standard library only).

## Download

**Option 1 — curl:**

```bash
curl -O https://raw.githubusercontent.com/hershyz/super-blotto/refs/heads/main/client/client.py
```

**Option 2 — GitHub Releases:**

Download from the [Releases](https://github.com/hershyz/super-blotto/releases) tab on the GitHub repo.

## Run

```bash
python3 client.py
```
